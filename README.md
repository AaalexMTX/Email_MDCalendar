# email_mdCalendar
定时（月度）邮件推送当月markdown版，可导入日历表格


## config 配置参数设置
| 参数           | 含义      | 示例            |
| ------------ | ------- |---------------|
| From         | 送件人邮箱号  | xx@163.com    |
| To           | 收件人邮箱号  | yy@163.com    |
| Auth_code    | 授权码     | 生成方式见资料       |
| email_server | 邮件服务器   | 默认stmp.qq.com |
| server_port  | 邮件服务器端口 | 默认 465        |
